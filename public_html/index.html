Scirerama

Nem todos os usuários de ontologias e/ou wikis semânticos são capazes de escrever ou entender os conceitos complexos que podem figurar em consul- tas do Semantic MediaWiki.

Como um dos objetivos da web semântica é tornar o conhecimento ampla- mente acessível, apresentamos o Scirerama uma linguagem visual para repre- sentar consultas do Semantic MediaWiki.

Elementos do Scirerama para representar um conceito

Sujeito

Representa pesquisas de categorias, páginas e nomespace.
Onde para cada tipo de sujeito uma imagem diferente esta associada e um rótulo deverá ser preenchido para validar a consulta.

Proprieade

Representa pesquisas de propriedade.
Onde a propriedade e o rótulo deverá ser preenchido para validar a consulta. Porém, no caso do rótulo, somente poderá ser fornecido um valor alfanumérico ou um sujeito.

E

Representa uma conjunção de consultas, que é composto por no mínimo dois casos.

OU

Representa uma disjunção de consultas, assim como E, é composto por no mínimo dois casos.

ANY

É um valor especial, i.e. predefinido, que pode ser usado no rótulo do sujeito ou no valor de propriedade.
